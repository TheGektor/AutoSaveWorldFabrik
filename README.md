# AutoSaveWorldFabrik

Серверный мод для Minecraft 1.20.1 на Fabric 0.16.14, который обеспечивает автоматическое сохранение мира и перезапуск сервера.

## Функции

- **Автосохранение**: Автоматически выполняет команду `save-all` через заданные интервалы
- **Автоперезапуск**: Поддерживает два режима перезапуска:
  - По времени (например, каждый день в полночь по МСК)
  - По интервалу (например, каждые 5 часов)

## Установка

1. Скачайте Fabric Loader 0.16.14 для Minecraft 1.20.1
2. Поместите файл мода в папку `mods` вашего сервера
3. Настройте конфигурацию в файле `autosaveworldfabrik.properties`

## Конфигурация

Создайте файл `autosaveworldfabrik.properties` в папке `config` сервера:

```properties
# Автосохранение каждые 5 минут (0 для отключения)
save-all-time=5

# Автоперезапуск
# Формат: -HH:MM для конкретного времени по МСК (например, -00:00 для полуночи)
# Формат: число для интервала в часах (например, 5 для каждых 5 часов)
# 0 для отключения
auto-restart-time=0
```

### Примеры конфигурации

**Автосохранение каждые 10 минут, перезапуск каждый день в 3:00 МСК:**
```properties
save-all-time=10
auto-restart-time=-03:00
```

**Автосохранение каждые 15 минут, перезапуск каждые 6 часов:**
```properties
save-all-time=15
auto-restart-time=6
```

**Только автосохранение каждые 5 минут:**
```properties
save-all-time=5
auto-restart-time=0
```

## Сборка

```bash
./gradlew build
```

Готовый мод будет в папке `build/libs/`.

## Требования

- Minecraft 1.20.1
- Fabric Loader 0.16.14+
- Fabric API
